# 🌎 Smart Travel Assistant

A comprehensive terminal-based travel planning application that combines multiple APIs to provide intelligent route planning, natural language processing, and travel assistance features.

## ✨ Features

- 🗺️ **Advanced Route Planning**
  - Multi-stop journeys
  - Alternative routes
  - Avoid highways, tolls, and ferries
  - Time-based routing

- 💬 **Natural Language Processing**
  - Convert conversational queries to routes
  - GPT-powered trip summaries
  - Intelligent query parsing

- 🎤 **Voice Features**
  - Text-to-speech navigation
  - Voice input support
  - Step-by-step audio directions

- 📊 **Smart Travel Features**
  - Fuel cost calculator
  - Weather integration
  - Points of interest finder
  - Interactive map generation

- ⭐ **Favorites Management**
  - Save frequent locations
  - Quick access with aliases
  - JSON-based storage
